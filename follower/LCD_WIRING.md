# ACM0802C-NLW-BBH LCD 配線ガイド

## LCD仕様

- 8文字 x 2行
- HD44780互換コントローラ
- 4ビットモード接続

## 配線表

| LCDピン | 名称 | 接続先 |
|---------|------|--------|
| 1       | VSS  | GND    |
| 2       | VDD  | 5V     |
| 3       | VO   | 可変抵抗（コントラスト調整） |
| 4       | RS   | A0     |
| 5       | RW   | GND    |
| 6       | E    | A1     |
| 7       | D0   | 未接続 |
| 8       | D1   | 未接続 |
| 9       | D2   | 未接続 |
| 10      | D3   | 未接続 |
| 11      | D4   | A2     |
| 12      | D5   | A3     |
| 13      | D6   | A4     |
| 14      | D7   | A5     |

## コントラスト調整

VO（3番ピン）に10kΩ可変抵抗を接続：

```
5V ─────┬───── 可変抵抗 ─────┬───── GND
        │                    │
        └── VO（LCDピン3）───┘
             （中央端子）
```

可変抵抗を回して文字が見えるように調整してください。

## 配線図

```
Arduino Uno          ACM0802C-NLW-BBH
─────────────        ─────────────────
    5V  ─────────────  VDD (2)
   GND  ─────────────  VSS (1)
   GND  ─────────────  RW  (5)
    A0  ─────────────  RS  (4)
    A1  ─────────────  E   (6)
    A2  ─────────────  D4  (11)
    A3  ─────────────  D5  (12)
    A4  ─────────────  D6  (13)
    A5  ─────────────  D7  (14)

可変抵抗（10kΩ）
    5V  ───[////]─── GND
            │
           VO (3)
```

## 表示内容

```
L:12.3
R:45.6
```

- 1行目: 左センサ距離 (cm)
- 2行目: 右センサ距離 (cm)
